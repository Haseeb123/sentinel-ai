# SentinelAI

**SentinelAI** is a Governance Runtime for Safe AI Agents.

Instead of allowing autonomous agents to directly execute tools, SentinelAI introduces a governance layer that evaluates every planned action before execution.

The project demonstrates modern AI engineering concepts including:

- Multi-Agent Architecture
- Governance Runtime
- Risk Assessment
- Policy Enforcement
- Audit Logging
- Tool Execution
- MCP Integration
- Google ADK Integration
- Memory
- Observability

---

## Why SentinelAI?

Current AI agent frameworks focus on making agents more capable.

SentinelAI focuses on making agents more trustworthy.

It introduces a runtime that evaluates every action before execution, allowing organizations to safely deploy autonomous AI systems.
